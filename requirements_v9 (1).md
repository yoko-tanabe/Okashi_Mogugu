## 1. サービス概要

### アプリのコンセプト

旅行者が**すれ違った人**と出会えるマッチングアプリ。リアルタイムではなく「5分前にすれ違った人」が表示される設計で、ストーカーリスクを下げながら一期一会の出会いを提供する。

### 一言で言うと

**「すれ違いを、出会いに変えるアプリ」**

### ターゲットユーザー

| ユーザー | ニーズ |
|---------|--------|
| 一人旅ユーザー | 現地で話し相手・観光仲間を探したい |
| バックパッカー | 世界中で出会いを集めたい |
| ゆるい交流希望者 | SNSより気軽・マッチングアプリより安全な出会いがほしい |

## 2. 核となる仕組み（重要）

### すれ違い〜マッチングの全体フロー

```
① ユーザーAとBが半径50m以内ですれ違う
        ↓
② その場所を離れる（同じ場所にいなくなる）
        ↓
③ 5分間が経過する
        ↓
④ お互いのカードがTinder風に表示される
   （共通点が多い人が上位に表示）
        ↓
⑤ AがBのカードを右スワイプ →「会いたい」
        ↓
⑥ Bに「会いたいリクエスト」通知が届く
        ↓
⑦ BがAのカードを確認し右スワイプで承認
        ↓
⑧ マッチング成立 → チャット画面が開放される
        ↓
⑨ チャットで待ち合わせ場所・時間を決める
        ↓
⑩「会う約束を確定」ボタンを両者が押す
        ↓
⑪ 約束時間から60分のカウントダウン開始
        ↓
⑫ 時間内に「会えた！」ボタンを押す
   → 徳ポイント獲得・パスポートにスタンプ付与
        ↓
⑬ 60分経過 → チャット自動終了・履歴削除
```

### カードが表示されない条件

以下のいずれかに該当する場合、相手のカードは表示されない。

| 条件 | 理由 |
|------|------|
| すれ違いから5分未満 | 今まだ近くにいる可能性があるため |
| 現在も同じ場所・店舗にいる | ストーカーリスク防止 |
| 「会いたくない」モードがON | ユーザーの意思尊重 |
| すでにブロックした相手 | 安全設計 |
| 性別条件が一致しない | フィルター設定 |
| 年齢条件が一致しない | フィルター設定 |
| すれ違いから24時間が経過した | 鮮度維持・一期一会の演出 |

## 3. カード表示順（優先度）

共通点が多い人ほど上位に表示される。

```
1位 ── 自由記述ワードが一致（推し名・アニメ名など）
2位 ── 趣味タグが3個以上一致
3位 ── 趣味タグが1〜2個一致
4位 ── 共通点ゼロ（最下位に表示・非表示にはしない）
```

共通点がある場合はカードに以下のようにハイライト表示される。

- 例：「🔥 鬼滅の刃が一致！」
- 例：「🎵 EDM・アニメが一致！」

## 4. 機能要件

### Must（ハッカソンで必ず実装する機能）

| 機能名 | 内容 |
|--------|------|
| すれ違い検知 | 半径50m以内に一定時間いたユーザーを記録する |
| 5分ディレイ表示 | 離れてから5分後にTinder風カードで表示する |
| 同一場所除外 | 現在も同じ場所にいる相手は表示しない |
| スワイプUI | 右「会いたい」／左「パス」 |
| 会いたくないモード | ON中はすれ違い記録停止・カード非表示 |
| プロフィール | 名前・国籍・性別・年代・趣味タグ・話せる言語・徳バッジ |
| 性別フィルター | 出会いたい相手の性別を設定・両者一致で表示 |
| 年齢フィルター | 出会いたい相手の年齢範囲を設定・両者一致で表示 |
| 徳システム | 行動に応じて徳ポイント付与・称号変化 |
| パスポート風スタンプ帳 | 会った人の国籍が入国スタンプ風に記録される |

### Should（あると完成度が上がる機能）

| 機能名 | 内容 |
|--------|------|
| マッチング後チャット | 両者マッチで待ち合わせ相談ができる |
| すれ違い場所の表示 | 「○○駅付近ですれ違い」とカードに表示 |
| 共通タグ・ワードハイライト | 一致した趣味・自由記述をカードに強調表示 |
| 安全機能 | ブロック・通報ができる |

### Nice to have（余裕があれば実装する機能）

| 機能名 | 内容 |
|--------|------|
| 多言語対応 | プロフィール・チャットの自動翻訳 |
| 旅友達ランキング | 何カ国の人と会ったかのランキング |

## 5. プロフィール設計

### 入力項目一覧

| 項目 | 内容 | 公開範囲 |
|------|------|---------|
| 名前 | ニックネームのみ・実名不要 | 公開 |
| 国籍 | 国旗アイコンで表示 | 公開 |
| 性別 | 男性／女性／その他／回答しない | 公開 |
| 年齢 | 生年月日を入力・表示は「20代」など年代のみ | 年代のみ公開 |
| 趣味タグ | 下記17個から最低3つ選択必須 | 公開 |
| 自由記述 | 推し・アニメ・キャラ名などを自由入力 | 公開 |
| 動画リンク | お気に入り動画URL（最大5本） | 公開 |
| 話せる言語 | 複数選択可 | 公開 |
| 旅のスタイル | 観光／グルメ／アドベンチャーなど | 公開 |
| 出会いたい性別 | 男性／女性／その他／こだわらない（複数選択可） | 非公開（マッチング処理のみ） |
| 出会いたい年齢範囲 | 上◯歳・下◯歳まで or こだわらない | 非公開（マッチング処理のみ） |

### 趣味タグ一覧（確定・17個）

| カテゴリ | タグ |
|---------|------|
| 🎌 アニメ・マンガ・ゲーム | アニメ / ゲーム / マンガ / コスプレ |
| 🎤 音楽・エンタメ | カラオケ / 男性アイドル / 女性アイドル / J-POP / K-POP / ジャズ / EDM / クラブ |
| 🍻 グルメ・ライフスタイル | bar/居酒屋 / コスメ / サウナ |
| ⛷️ アウトドア・スポーツ | スキー/ボード / 登山 |

### 自由記述エリア

- 好きなアニメ・キャラクター名・推しのアイドル・アーティスト名など何でも入力可能
- 例：「鬼滅の刃・煉獄さん・星野リン」
- 相手の自由記述と一致するワードがあればカードにハイライト表示される

### お気に入り動画リンク

- URLをペーストするとOGP情報を自動取得
- LINEのリンクプレビューと同様のUIで表示（サムネイル・タイトル・ドメイン名）
- 最大5本まで登録可能
- 対応サービス：YouTube / TikTok / Instagram / Vimeo / ニコニコ動画

## 6. 徳システム設計

### 徳ポイントが貯まる条件

| アクション | 獲得ポイント |
|-----------|------------|
| 実際に会う（「会えた！」ボタンを押す） | +30 |
| 新しい国の人と初めて会う | +20 |
| プロフィールを充実させる（初回） | +10 |
| 節目の達成（10カ国達成など） | +100 |

### 徳レベルと称号

```
0〜99点    ： 旅のたまご  🥚
100〜299点 ： 旅人        🎒
300〜599点 ： 渡り鳥      🐦
600〜999点 ： 世界の旅人  🌍
1000点〜   ： 伝説の旅人  ✨
```

### 徳の効果

- プロフィール画面・スワイプカードに称号バッジが表示される
- 徳が高いユーザーは信頼の証として、相手がスワイプする際の判断材料になる

## 7. パスポート風スタンプ帳設計

### デザインイメージ

- **表紙** ： 自分の国旗カラーをベースにしたパスポート風デザイン。名前・徳レベル・訪問国数を表示
- **中ページ** ： 会った人の国籍ごとに入国スタンプ風のデザインで記録。スタンプには日付・すれ違い場所が入る
- **ページ送り** ： ページをめくるアニメーション付き
- **パスポート更新** ： 全ページが埋まったら「新しいパスポート」に切り替わる達成感の演出
- **SNSシェア** ： スタンプ帳の画面をそのままシェアできるボタンを設置

## 8. 1時間限定の出会い設計

```
マッチング成立 → チャット開放
        ↓
チャットで待ち合わせ場所・時間を相談
        ↓
「会う約束を確定」ボタンを両者が押す
（片方だけでは発動しない）
        ↓
約束時間から60分のカウントダウン開始
（チャット画面に常時タイマー表示）
        ↓
時間内に「会えた！」を押す
→ 徳+30・パスポートにスタンプ付与

時間切れまでに押さない
→ 徳・スタンプなし
        ↓
60分経過 → チャット自動終了・延長なし・履歴削除
```

## 9. 安全設計

| 設計 | 目的 |
|------|------|
| 5分ディレイ | 「今ここにいる」情報を相手に渡さない |
| 同一場所除外 | ストーカー行為の防止 |
| 実名不要 | プライバシー保護 |
| 年代表示のみ | 正確な年齢の非公開 |
| 会いたくないモード | ワンタップで即座にON/OFF |
| チャット履歴削除 | 60分後に自動削除 |
| ブロック・通報 | 不審なユーザーへの対処 |
